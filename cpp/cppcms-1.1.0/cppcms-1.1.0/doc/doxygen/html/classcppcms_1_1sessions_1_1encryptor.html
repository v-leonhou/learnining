Detailed Description

This is an interface to generic session cookies encryption or signing API.

Note for users implementing their own ecryptor classes:

  • Be super paranoid and extremely careful in what you are doing.
  • Take in mind that this object are created and destroyed much frequently then they are actually accessed, so lazy initialization is your best friend.

Note this class does not have to be thread safe to use from multiple threads.
